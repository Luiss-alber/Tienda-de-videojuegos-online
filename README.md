# Tienda-de-videojuegos-online
Este proyecto consiste en un Análisis Exploratorio de Datos (EDA) sobre la industria de los videojuegos, con el objetivo de identificar patrones, tendencias y factores que influyen en las ventas globales.

A través del análisis, se busca responder preguntas como:

¿Qué géneros son más populares?
¿Qué plataformas dominan el mercado?
¿Cómo han evolucionado las ventas a lo largo del tiempo?
¿Existen diferencias entre regiones?

## Objetivos
Analizar el comportamiento del mercado de videojuegos
Identificar tendencias por género, plataforma y año
Detectar posibles correlaciones entre variables
Generar insights para la toma de decisiones

## Dataset

El dataset contiene información sobre:

Nombre del videojuego
Plataforma
Año de lanzamiento
Género
Ventas globales y por región (NA, EU, JP, Other)
Puntuación de críticos profesionales 
Puntuación de usuarios 
Clasificación
